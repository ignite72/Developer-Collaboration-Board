# Developer Collaboration Board

A starter full-stack project for managing software team tasks, assignees, delivery status, and project progress.

## Stack

- Frontend: React + Vite
- Backend: Node.js + Express
- Future database options: MongoDB or Supabase

## Features in this starter

- Dashboard cards for team task stats
- Create task form with assignee, status, and priority
- Search and filter by workflow status
- Sample collaboration board
- Express API with `GET /api/tasks` and `POST /api/tasks`

## Project structure

```text
job-tracker-app/
  frontend/
  backend/
```

## Run locally

### Frontend

```bash
cd frontend
npm install
npm run dev
```

### Backend

```bash
cd backend
npm install
npm run dev
```
