# Developer Collaboration Board

A MERN-stack team collaboration platform for managing software projects — create projects, 
assign tasks, track progress, and keep teamwork visible in one focused workspace.

> 🚧 **Status:** In active development. Core features are being built out; not yet deployed.

## Features

- User authentication (signup/login)
- Project and task creation
- Task assignment across team members
- Centralized board view for tracking progress

## Tech Stack

**Frontend:** React, Vite  
**Backend:** Node.js, Express.js  
**Database:** MongoDB
